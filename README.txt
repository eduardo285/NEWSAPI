Project Title: NewZone

Project Description: NewZone is an android based news application that provides world news trend. The application is made through API integration specifically NewsAPI. It has the ability to deliver top daily world news from over 5,000 leading news sources and blogs. Every article shows its time published, title, pictures and the news itself.This application will make people up-to-date about the events as they occur in different places of the world.
